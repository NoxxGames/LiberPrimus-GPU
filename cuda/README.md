# cuda

## Purpose

Optional CUDA smoke scaffold and future GPU acceleration home.

## What belongs here

Guarded CUDA code, CUDA parity tests, CUDA benchmark documentation, and future kernels after CPU references exist.

## What does not belong here

Cryptanalysis kernels without CPU references, profiling output, generated result dumps, or long-running benchmark output.

## Codex modification policy

Codex may edit guarded CUDA scaffolding and docs. Future kernels require parity tests and benchmark plans.

## Stage 0A restrictions

Only the optional smoke kernel is allowed.
