# Token-Block Coordinate CLI

Stage 5AR extends `libreprimus token-block` with original-image coordinate-lock commands.

Typical local sequence:

```powershell
.\.venv\Scripts\python.exe -m libreprimus.cli token-block build-stage5ar-original-image-source-lock --search-roots third_party research-inputs website-export data --stage5ap-image-provenance data/token-block/stage5ap-page49-51-image-provenance.yaml --results-dir experiments/results/token-block/stage5ar --out-source-lock data/token-block/stage5ar-original-page-image-source-lock.yaml --out-variants data/token-block/stage5ar-original-page-image-variants.yaml
.\.venv\Scripts\python.exe -m libreprimus.cli token-block build-stage5ar-page-split --stage5ap-transcription data/token-block/stage5ap-token-block-canonical-transcription.yaml --original-image-source-lock data/token-block/stage5ar-original-page-image-source-lock.yaml --results-dir experiments/results/token-block/stage5ar --out-policy data/token-block/stage5ar-page-split-policy.yaml --out-records data/token-block/stage5ar-page-split-records.yaml
.\.venv\Scripts\python.exe -m libreprimus.cli token-block build-stage5ar-pixel-coordinates --stage5ap-transcription data/token-block/stage5ap-token-block-canonical-transcription.yaml --stage5ap-logical-coordinates data/token-block/stage5ap-token-block-coordinate-records.yaml --original-image-source-lock data/token-block/stage5ar-original-page-image-source-lock.yaml --page-split-records data/token-block/stage5ar-page-split-records.yaml --results-dir experiments/results/token-block/stage5ar --out-policy data/token-block/stage5ar-token-pixel-coordinate-policy.yaml --out-records data/token-block/stage5ar-token-pixel-coordinate-records.yaml
.\.venv\Scripts\python.exe -m libreprimus.cli token-block build-stage5ar-case-policy --stage5ap-transcription data/token-block/stage5ap-token-block-canonical-transcription.yaml --pixel-coordinate-records data/token-block/stage5ar-token-pixel-coordinate-records.yaml --results-dir experiments/results/token-block/stage5ar --out-policy data/token-block/stage5ar-token-case-policy.yaml --out-ambiguities data/token-block/stage5ar-token-case-ambiguity-records.yaml
.\.venv\Scripts\python.exe -m libreprimus.cli token-block validate-stage5ar-coordinates --original-image-source-lock data/token-block/stage5ar-original-page-image-source-lock.yaml --page-split-records data/token-block/stage5ar-page-split-records.yaml --pixel-coordinate-policy data/token-block/stage5ar-token-pixel-coordinate-policy.yaml --pixel-coordinate-records data/token-block/stage5ar-token-pixel-coordinate-records.yaml --case-policy data/token-block/stage5ar-token-case-policy.yaml --case-ambiguities data/token-block/stage5ar-token-case-ambiguity-records.yaml --results-dir experiments/results/token-block/stage5ar --out data/token-block/stage5ar-token-coordinate-validation.yaml
.\.venv\Scripts\python.exe -m libreprimus.cli token-block validate-stage5ar --original-source-lock data/token-block/stage5ar-original-page-image-source-lock.yaml --variants data/token-block/stage5ar-original-page-image-variants.yaml --page-split-policy data/token-block/stage5ar-page-split-policy.yaml --page-split-records data/token-block/stage5ar-page-split-records.yaml --pixel-coordinate-policy data/token-block/stage5ar-token-pixel-coordinate-policy.yaml --pixel-coordinate-records data/token-block/stage5ar-token-pixel-coordinate-records.yaml --case-policy data/token-block/stage5ar-token-case-policy.yaml --case-ambiguities data/token-block/stage5ar-token-case-ambiguity-records.yaml --coordinate-validation data/token-block/stage5ar-token-coordinate-validation.yaml --source-lock-update data/token-block/stage5ar-token-block-source-lock-update.yaml --null-control-update data/token-block/stage5ar-token-block-null-control-update.yaml --dwh-context data/token-block/stage5ar-dwh-coordinate-context.yaml --guardrail data/token-block/stage5ar-guardrail.yaml --next-stage-decision data/project-state/stage5ar-next-stage-decision.yaml --summary data/project-state/stage5ar-summary.yaml --results-dir experiments/results/token-block/stage5ar
```

Generated JSON/JSONL reports under `experiments/results/token-block/stage5ar/` remain ignored. The CLI reads original local images for metadata and deterministic coordinate projections only. It must not perform OCR, AI/ML interpretation, semantic image interpretation, hidden-content forensics, stego extraction, hash/preimage search, decoding, CUDA execution, benchmarking, or scored experiments.
