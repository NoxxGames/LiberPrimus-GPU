# Token-Block Preflight Runner Scaffold CLI

Stage 5BB extends `libreprimus token-block` with no-execution scaffold commands.

Typical local sequence:

```powershell
python -m libreprimus.cli token-block build-stage5bb-active-manifest-registry --results-dir experiments/results/token-block/stage5bb --out-registry data/token-block/stage5bb-active-manifest-registry.yaml --out-precedence-policy data/token-block/stage5bb-manifest-precedence-policy.yaml
python -m libreprimus.cli token-block audit-stage5bb-legacy-pointers --active-registry data/token-block/stage5bb-active-manifest-registry.yaml --precedence-policy data/token-block/stage5bb-manifest-precedence-policy.yaml --stage5az-execution-gates data/token-block/stage5az-repaired-execution-gates.yaml --stage5ay-variant-family data/token-block/stage5ay-bounded-variant-family-manifest.yaml --stage5az-variant-family data/token-block/stage5az-repaired-bounded-variant-family-manifest.yaml --results-dir experiments/results/token-block/stage5bb --out data/token-block/stage5bb-legacy-pointer-audit.yaml
python -m libreprimus.cli token-block validate-stage5bb-manifest-references --active-registry data/token-block/stage5bb-active-manifest-registry.yaml --precedence-policy data/token-block/stage5bb-manifest-precedence-policy.yaml --legacy-pointer-audit data/token-block/stage5bb-legacy-pointer-audit.yaml --results-dir experiments/results/token-block/stage5bb --out-reference-validation data/token-block/stage5bb-manifest-reference-validation.yaml --out-branch-eligibility-validation data/token-block/stage5bb-branch-eligibility-reference-validation.yaml
python -m libreprimus.cli token-block validate-stage5bb --active-registry data/token-block/stage5bb-active-manifest-registry.yaml --precedence-policy data/token-block/stage5bb-manifest-precedence-policy.yaml --legacy-pointer-audit data/token-block/stage5bb-legacy-pointer-audit.yaml --reference-validation data/token-block/stage5bb-manifest-reference-validation.yaml --branch-eligibility-validation data/token-block/stage5bb-branch-eligibility-reference-validation.yaml --loader-policy data/token-block/stage5bb-loader-scaffold-policy.yaml --runner-manifest data/token-block/stage5bb-runner-scaffold-manifest.yaml --dry-run-preview data/token-block/stage5bb-dry-run-plan-preview.yaml --branch-counter data/token-block/stage5bb-branch-counter-summary.yaml --family-summary data/token-block/stage5bb-family-enumeration-summary.yaml --gate-policy data/token-block/stage5bb-execution-gate-enforcement-policy.yaml --gate-validation data/token-block/stage5bb-execution-gate-validation.yaml --fixture-policy data/token-block/stage5bb-result-schema-fixture-policy.yaml --fixture-records data/token-block/stage5bb-fixture-result-schema-records.yaml --validation-evidence data/token-block/stage5bb-validation-evidence-index.yaml --no-execution-proof data/token-block/stage5bb-no-execution-proof.yaml --dwh-context data/token-block/stage5bb-dwh-runner-context.yaml --guardrail data/token-block/stage5bb-guardrail.yaml --next-stage data/project-state/stage5bb-next-stage-decision.yaml --summary data/project-state/stage5bb-summary.yaml --results-dir experiments/results/token-block/stage5bb
```

The commands write compact committed YAML records and ignored local JSON previews. They do not generate real token-block bytes, run variants, compare hashes, decode, score, run CUDA, benchmark, or make solve claims.
