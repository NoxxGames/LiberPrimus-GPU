# Token Case Review Pack CLI

Stage 5AT extends `libreprimus token-block` with case-review pack commands.

Typical local sequence:

```powershell
.\.venv\Scripts\python.exe -m libreprimus.cli token-block repair-stage5at-variant-classifier --out data/token-block/stage5at-variant-classifier-repair-summary.yaml
.\.venv\Scripts\python.exe -m libreprimus.cli token-block build-stage5at-case-review-policy --stage5ar-case-policy data/token-block/stage5ar-token-case-policy.yaml --stage5ar-case-ambiguities data/token-block/stage5ar-token-case-ambiguity-records.yaml --out data/token-block/stage5at-case-review-policy.yaml
.\.venv\Scripts\python.exe -m libreprimus.cli token-block build-stage5at-case-challenge-set --stage5ap-transcription data/token-block/stage5ap-token-block-canonical-transcription.yaml --stage5ar-pixel-coordinates data/token-block/stage5ar-token-pixel-coordinate-records.yaml --stage5ar-case-ambiguities data/token-block/stage5ar-token-case-ambiguity-records.yaml --policy data/token-block/stage5at-case-review-policy.yaml --out-challenges data/token-block/stage5at-case-review-challenge-set.yaml --out-canonical data/token-block/stage5at-canonical-transcription-challenge-set.yaml
.\.venv\Scripts\python.exe -m libreprimus.cli token-block build-stage5at-decision-template --challenges data/token-block/stage5at-case-review-challenge-set.yaml --out data/token-block/stage5at-human-review-decision-template.yaml
.\.venv\Scripts\python.exe -m libreprimus.cli token-block build-stage5at-review-pack --challenges data/token-block/stage5at-case-review-challenge-set.yaml --canonical-challenges data/token-block/stage5at-canonical-transcription-challenge-set.yaml --decision-template data/token-block/stage5at-human-review-decision-template.yaml --original-image-source-lock data/token-block/stage5ar-original-page-image-source-lock.yaml --review-pack-root human-review-packs/stage5at/token-case-review --results-dir experiments/results/token-block/stage5at --out-crop-manifest data/token-block/stage5at-case-review-crop-manifest.yaml --out-pack-manifest data/token-block/stage5at-case-review-pack-manifest.yaml
.\.venv\Scripts\python.exe -m libreprimus.cli token-block validate-stage5at --case-review-policy data/token-block/stage5at-case-review-policy.yaml --case-challenges data/token-block/stage5at-case-review-challenge-set.yaml --canonical-challenges data/token-block/stage5at-canonical-transcription-challenge-set.yaml --crop-manifest data/token-block/stage5at-case-review-crop-manifest.yaml --decision-template data/token-block/stage5at-human-review-decision-template.yaml --pack-manifest data/token-block/stage5at-case-review-pack-manifest.yaml --variant-repair data/token-block/stage5at-variant-classifier-repair-summary.yaml --doc-drift-summary data/token-block/stage5at-doc-drift-repair-summary.yaml --null-control-update data/token-block/stage5at-null-control-case-update.yaml --dwh-case-context data/token-block/stage5at-dwh-case-context.yaml --guardrail data/token-block/stage5at-guardrail.yaml --next-stage-decision data/project-state/stage5at-next-stage-decision.yaml --summary data/project-state/stage5at-summary.yaml --review-pack-root human-review-packs/stage5at/token-case-review --results-dir experiments/results/token-block/stage5at
```

The build commands may read local ignored page images only to create deterministic crops. Validation works from committed metadata and guardrails. Generated packs and reports remain ignored.
