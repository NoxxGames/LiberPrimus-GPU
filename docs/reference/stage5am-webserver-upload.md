# Stage 5AM Webserver Upload

Upload this directory if you need the private static research index:

```text
website-export/stage5am/research-index/
```

Do not upload:

- `third_party/`
- `data/raw/`
- `research-inputs/`
- `experiments/results/`
- `codex-output/`
- local Deep Research reports

The export is metadata-only and review-gated. Keep it behind private access controls if it is placed on a server. The renderer does not make records public-ready; publication gates in the data still control that decision.
