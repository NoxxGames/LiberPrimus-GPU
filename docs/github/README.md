# GitHub Project Support

This folder contains repository-owned GitHub support files: workflow policy, label definitions, issue seed files, wiki source pages, and bootstrap reports.

The repository documentation and tutorials are the source of truth. GitHub wiki pages are a public mirror for convenience and must not contain raw corpus dumps, generated alignment records, or solve claims.

Use scripts under `scripts/github/` to verify the target repository, create labels and issues idempotently, and publish wiki pages.
