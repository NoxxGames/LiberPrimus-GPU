# Document data provenance, licensing, and redistribution policy

## Summary

Expand public policy for data provenance, licensing, redistribution, and local acquisition instructions.

## Current Status

Raw files are ignored and hash-locked where present.

## Scope

Clarify what contributors may commit, how to cite sources, and how to document local-only raw files.

## Non-Goals

Do not redistribute raw corpus files without review.

## Deliverables

Updated `DATASET.md`, tutorials, issue templates, and source metadata examples.

## Acceptance Criteria

New contributors can understand how to work without committing restricted raw data.

## Safety/Provenance Rules

Prefer acquisition instructions and metadata locks over committed raw files.

Suggested labels: data-provenance, documentation, safety, needs-human-review

## Dependencies

Ongoing source review.
