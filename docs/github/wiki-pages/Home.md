# LiberPrimus GPU

LiberPrimus GPU is a C++20/Python/optional-CUDA workbench for reproducible Liber Primus cryptanalysis research.

Current status: Stage 0D is complete and Stage 0D-followup is needed. No canonical corpus is active, no unsolved page is claimed solved, and CUDA is not used by the current parser/alignment stages.

Repository tutorials and docs are the source of truth. This wiki is a public mirror and entry point.
