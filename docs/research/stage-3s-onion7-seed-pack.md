# Stage 3S Onion 7 Seed Pack

Stage 3S is the first bounded execution stage after the Stage 3R post-Discord manifest queue.

## Research Question

Can the explicit Onion 7 number table and reviewed derived tables produce a signal when used as bounded mod-29 subtract streams over the same reviewable text slice used in prior Stage 3 text experiments?

## Method

The executor validates `EXP-3R-003`, loads the reviewed Onion 7 raw and derived value spaces, enumerates manifest-declared routes, directions, and reset modes, reduces numeric streams mod 29, and applies:

```text
output_index = (cipher_index - stream_value[position]) mod 29
```

Only transformable rune/index tokens advance the stream. Separator tokens are preserved. Line-reset candidates require line metadata and otherwise emit explicit warnings.

## Result

The local run executed `72` candidates under the `144` cap and deferred `0`.

Top candidate:

- value space: `raw_table`
- route: `row_major`
- direction: `reverse`
- reset mode: `none`
- score: `1.460714`
- calibrated confidence: `inconclusive`

The result is consistent with prior noisy or inconclusive Stage 3 text probes. It is a review lead at most, not solve evidence.

## Policy Notes

Stage 3S does not execute cookie or GP/rune manifests, does not process Discord logs or page images, does not use CUDA, does not activate canonical corpus, does not finalize page boundaries, and does not claim a solve.
