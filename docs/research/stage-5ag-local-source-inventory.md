# Stage 5AG Local Source Inventory

Stage 5AG joins Stage 5AF source manifests to local ignored material now present under `third_party/`. The result is a compact source-lock inventory ledger, not source interpretation.

The local source inventory found enough material to justify a curated extraction-prep stage, but not enough to hand off directly to Deep Research. Five research bundles are partially ready and five are not ready. Stage 5AH should curate/extract bounded metadata from matched local material into ignored research-input bundles while committing only compact extraction metadata.

The stage keeps the following boundaries explicit:

- Local archive listings are inventory metadata only.
- Unclassified local sources require manual classification.
- Missing manifest sources remain source-lock gaps.
- Google Drive is not project storage.
- Network fetches and online clones require a later explicit stage.
- Source-lock inventory does not authorize CUDA, scored experiments, website expansion, or solve claims.
