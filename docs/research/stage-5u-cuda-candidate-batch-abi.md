# Stage 5U CUDA Candidate Batch ABI Research Note

Stage 5U closes the Stage 5T ABI gaps at the contract level.

It records Candidate Batch ABI v0, including token buffers, transform parameter surfaces, explicit-key schedules, prime-stream schedules, Stage 4I-compatible score vectors, deterministic top-k metadata, backend ownership surfaces, and Stage 4P-compatible result-store summaries.

All Stage 5T ABI gaps are `closed_by_contract`, but implementation remains pending. The next bounded step is a native no-GPU reference adapter and conformance fixture stage, not more CUDA execution.

No new evidence of a solve is created.
