# Stage 5AU Token Case Review Pack V2

Stage 5AU converts user feedback about the Stage 5AT review pack into a durable usability-repair record. The Stage 5AT pack is preserved as a historical generated pack but is explicitly blocked for manual review because it exposed only part of the challenge set and did not surface enough context for reliable decisions.

The v2 pack uses deterministic crop geometry from Stage 5AR original-image coordinates. It displays the original cell crop, glyph-candidate crop, small/medium/large context crops, row context, page-strip context, and overlays for each case-review challenge. It also surfaces all canonical-transcription challenge records.

No token decisions are made by Stage 5AU. Stage 5AV has since integrated the explicit human decisions from the local filled template as compact metadata rather than using generated crops as automatic evidence.
