# Stage 5AJ UsefulFiles Integration

Stage 5AJ inserts UsefulFilesAndIdeas into the research handoff layer before the next Deep Research prompt. The new local files include two workbooks, an important-links text file, an ideas text file, and an image. The stage records hashes, sizes, workbook/sheet/cell summary metadata, URL indexes, source-card updates, and private bundle-readiness updates.

The main policy change is extraction fidelity. Private Deep Research material should preserve evidence-bearing technical detail instead of over-redacting it. Public website ingest remains conservative metadata only and stays disabled for publication.

Stage 5AJ selects Stage 5AK - Deep Research source inventory and reliability prompt. Stage 5AK should consume the committed Stage 5AI/5AJ manifests, source cards, content indexes, policy records, and readiness summaries, not raw local files.
