# docs/cuda

## Purpose

CUDA design and verification notes.

## What belongs here

Kernel design notes, parity requirements, memory-layout plans, and profiling policy.

## What does not belong here

Profiling output, stress-test logs, or generated benchmark data.

## Codex modification policy

Codex may edit docs. CUDA code must remain optional and test-backed.
