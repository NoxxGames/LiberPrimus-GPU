# Local Source Inventory Workflow

Use this workflow for Stage 5AG-style local source inventory, Stage 5AI-style curated extraction metadata, Stage 5AJ-style UsefulFilesAndIdeas metadata, Stage 5AK-style community-facts claim metadata, Stage 5AL-style website-ingest staging, Stage 5AM-style static index rendering, Stage 5AN-style private content-pack generation, Stage 5AP-style token-block source-lock metadata, Stage 5AR-style original-image coordinate-lock metadata, Stage 5AT-style token case-review pack metadata, and Stage 5BI-style Fandom/source-lock crosswalk metadata after the Stage 5AH doc-staleness gate is clean.

1. Put user-provided source files under an ignored local root, normally `third_party/`.
2. Confirm raw files are ignored with `git status --short third_party` and `git check-ignore -v`.
3. Run `libreprimus source-harvester inventory-local-sources`.
4. Link local paths to the Stage 5AF manifest with `link-local-sources`.
5. Build source-lock candidates, gap records, bundle readiness, guardrails, next-stage decision, and summary records.
6. Commit only compact metadata under `data/source-harvester/`, schemas, docs, tests, and source code.
7. Before Fandom/source-lock follow-up work, validate Stage 5AI, Stage 5AJ, Stage 5AK, Stage 5AL, Stage 5AM, Stage 5AN, Stage 5AP, Stage 5AR, Stage 5AT, Stage 5BF, and Stage 5BI records and confirm current docs name Stage 5BJ as the next stage.

Do not use Google Drive as a project storage location. Google/Dropbox/Colab/community exports must be local ignored files. Do not commit raw archives, images, PDFs, HTML, DOCX, XLSX, audio, video, extracted bodies, full generated inventories, generated research bundles, static website exports, human-review pack bodies, Fandom HTML/images, spreadsheet bodies, or `codex-output/**`. Stage 5AI generated bundle bodies under `research-inputs/stage5ai/`, Stage 5AJ generated UsefulFiles bodies under `research-inputs/stage5aj/`, Stage 5AK community-facts addenda under `research-inputs/stage5ak/`, Stage 5AL private Deep Research export helpers under `research-inputs/stage5al/`, Stage 5AM static index files under `website-export/stage5am/`, Stage 5AN private pack/site files under `deep-research-content-packs/stage5an/` and `website-export/stage5an/`, Stage 5AP token-block/stego-control reports under `experiments/results/token-block/stage5ap/` and `experiments/results/stego-controls/stage5ap/`, Stage 5AR coordinate reports under `experiments/results/token-block/stage5ar/`, and Stage 5AT review-pack files under `human-review-packs/stage5at/token-case-review/` are private/local inputs, not publication artefacts.
