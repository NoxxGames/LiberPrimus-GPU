# Historical Route Source-Lock Workflow

Stage 5BF consumes `third_party/CicadaSolversIddqd` as ignored local source material.

Workflow:

1. Locate the archive without network cloning.
2. Hash and inventory local files.
3. Classify high-priority route artefacts.
4. Record trust classes and source gaps.
5. Prepare Deep Research review metadata.

Do not stage raw archive files, generated full inventories, content packs, or `codex-output`.
