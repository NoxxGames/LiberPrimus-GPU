# Stage 5AM Static Research Index

Stage 5AM is not a cryptanalytic experiment. It is a static renderer and upload-package stage for Stage 5AL metadata.

Inputs:

- `data/website-ingest/stage5al/`
- `data/source-harvester/stage5al-summary.yaml`

Outputs:

- committed renderer schemas and compact records under `data/website-render/`
- ignored static export under `website-export/stage5am/research-index/`
- ignored validation reports under `experiments/results/website-render/stage5am/`

The stage performs no Deep Research, network fetch, source-gap closure, OCR, AI/ML interpretation, image/stego/audio processing, CUDA execution, benchmarking, scored experiment execution, method-status upgrade, canonical-corpus activation, page-boundary finalisation, or solve claim.
