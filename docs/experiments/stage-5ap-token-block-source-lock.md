# Stage 5AP Token-Block Source Lock

Stage 5AP is not an experiment. It is a source-lock and positive-control preflight stage for the page 49-51 token block.

Outputs:

- 15 page-image metadata lock records.
- 32 by 8 canonical token transcription with 256 tokens.
- 256 logical coordinate records.
- Primary-60 alphabet and byte-mapping preflight records.
- 5 null-control plan entries.
- DWH context status `review_only_external_deep_research_context`.
- OutGuess toolchain readiness, positive-control matrix, historical fixture blockers, and guardrails.

Stage 5AP does not run Deep Research, decode the block, search hashes, run OutGuess on LP pages, process raw page images, run CUDA, benchmark, execute scored experiments, or claim a solve. Generated reports remain ignored under `experiments/results/token-block/stage5ap/` and `experiments/results/stego-controls/stage5ap/`.
