# Stage 5U CUDA Candidate Batch ABI

Stage 5U is not an experiment. It defines metadata contracts for a future candidate batch ABI.

Committed outputs live under `data/cuda/` and include ABI, token-buffer, transform-parameter, key-schedule, stream-schedule, score-vector, top-k, backend-surface, result-store compatibility, ABI gap closure, next-stage decision, and summary records.

Ignored reports live under `experiments/results/cuda-candidate-batch-abi/stage5u/`.

Stage 5U runs no CUDA, no native/CUDA CMake build, no benchmarks, no unsolved pages, no raw data, no generated-body publication, no method-status upgrade, and no solve claim.
