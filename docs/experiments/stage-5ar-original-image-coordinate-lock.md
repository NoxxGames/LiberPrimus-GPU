# Stage 5AR Original-Image Coordinate Lock

Stage 5AR is not an experiment. It is a coordinate source-lock and review-preflight stage for the Stage 5AP page 49-51 token block.

Outputs:

- 3 selected original page images for pages 49, 50, and 51.
- 15 local page-image variant records.
- A source-backed 10/13/9 page-split policy.
- 256 token pixel-coordinate records in original-image pixel space.
- Coordinate validation status `valid_with_review_required`.
- 9 unresolved case-ambiguity classes.
- Source-lock, null-control, Deep Web Hash context, guardrail, next-stage, and summary records.

Generated coordinate reports remain ignored under `experiments/results/token-block/stage5ar/`. Raw page images remain ignored under `third_party/LiberPrimusPages/`.

Stage 5AR performs no Deep Research, network fetch, live scrape, online clone, OCR, AI/ML interpretation, semantic image interpretation, hidden-content image forensics, stego execution, OutGuess on LP pages, DWH hash/preimage search, decode attempt, hypothesis execution, CUDA, benchmarks, scored experiments, canonical-corpus activation, page-boundary finalisation, method-status upgrade, or solve-claim workflow.
