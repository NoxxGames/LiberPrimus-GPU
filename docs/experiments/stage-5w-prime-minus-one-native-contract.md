# Stage 5W Prime-Minus-One Native Contract

Stage 5W is a contract-preparation stage for prime-minus-one stream parity.

It creates source inventory, stream contract, prime schedule, Candidate Batch ABI mapping, native parity preparation, result-store preflight, guardrail, next-stage decision, and summary records. It writes ignored JSON reports under `experiments/results/prime-minus-one-native-contract/stage5w/`.

Stage 5W does not run CUDA, run native/CUDA CMake, benchmark, process raw data, execute unsolved pages, publish generated bodies, or make solve claims.
