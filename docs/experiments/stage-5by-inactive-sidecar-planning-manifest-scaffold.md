# Stage 5BY Inactive-Sidecar Planning Manifest Scaffold

Stage 5BY is not an experiment. It is a metadata-only planning scaffold for the String 4 inactive sidecar.

The stage consumes the Stage 5BX `accept_with_warnings` review, classifies Stage 5BW source-digest duplicate rows, adds source-digest unique-path validation, and maps Stage 5BW record-family filename equivalents for future review. It creates an inactive planning-sidecar scaffold and a no-execution planning-ingestion sidecar, but it does not activate String 4.

No active manifests are superseded. Stage 5BD dry-run plans and run-plan IDs remain unchanged. Generated diagnostics under `experiments/results/token-block/stage5by/` remain ignored.

Blocked actions remain blocked: byte-stream generation, variant materialisation, branch enumeration, DWH/hash/preimage search, decode, scoring, stego/audio/image/OCR/AI, CUDA, benchmarks, website expansion, method-status upgrades, canonical-corpus activation, page-boundary finalisation, and solve claims.
