# Historical Route Trust Classification

Stage 5BF trust classes describe provenance quality, not truth.

Local file hashes stabilize a local archive copy. They do not prove that a file is original, authentic, intentional, or solve evidence. PGP blocks are recorded as present but not verified online. Wiki and narrative route descriptions remain secondary until tied to exact local artefacts.

The committed trust policy lives at `data/historical-route/stage5bf-trust-classification-policy.yaml`.
