# Token Variant Branch Manifest

The Stage 5AV branch manifest is compact metadata for unresolved page 49-51 token-case ambiguity. It does not enumerate the full Cartesian product and does not generate variant byte streams.

## Contents

- Baseline: Stage 5AP canonical token block with Stage 5AV keep-current confirmations.
- Unresolved cases: 77 records with reviewer-declared possible tokens.
- Controls: baseline current-token, single-case option, randomized assignment, and unchanged-canonical controls.
- Primary-60 impact: mappable and unmappable option counts plus value deltas.

The branch upper bound is recorded as metadata only. Stage 5AW may design bounded preflight manifests from this compact branch surface, but Stage 5AV does not execute them.
