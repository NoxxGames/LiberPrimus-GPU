# Prime-Minus-One CUDA Validation Vectors

Stage 5Z defines seven validation-vector records:

- synthetic positive control;
- bounded p56 fixture-safe reference;
- full-p56 blocked control;
- invalid token value control;
- invalid stream-start control;
- separator-preservation control;
- zero-transformable-token control.

Only synthetic vectors are candidates for Stage 5AA execution. The bounded p56 vector is a contract reference, not Stage 5AA execution scope. Full p56 remains blocked until a complete committed token buffer is explicitly scoped.
