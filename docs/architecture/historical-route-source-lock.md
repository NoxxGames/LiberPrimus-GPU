# Historical Route Source Lock

Stage 5BF source-locks the local 2012-2017 Cicada route archive at `third_party/CicadaSolversIddqd`.

The stage records file paths, sizes, SHA-256 hashes, archive tree digest, route families, trust classes, and compact candidate records. It does not commit raw archive files or generated full inventories.

This is source-lock metadata only. It does not run historical techniques, stego tools, OutGuess, OpenPuff, MP3Stego, online PGP verification, OCR, AI/ML interpretation, DWH/hash search, token experiments, CUDA, benchmarks, or solve claims.
