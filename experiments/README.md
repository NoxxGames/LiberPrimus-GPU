# experiments

## Purpose

Manifest-driven experiment definitions and ignored output locations.

## What belongs here

YAML manifests, result/report README files, and later reviewed experiment metadata.

## What does not belong here

Committed generated result dumps, logs, databases, JSONL files, CSV files, or parquet files.

## Codex modification policy

Codex may add manifests and documentation. Generated outputs must remain ignored.

## Stage 0A restrictions

Only a non-cryptanalytic smoke manifest exists.
