# Scoring Data

This directory contains tiny committed scoring helper data and durable scoring-policy records for deterministic local triage. It must not become a large dictionary or a source of solve claims.

Stage 4I adds scorer inventory, confidence-label, compatibility-map, calibration-profile, and calibration-report records. These records describe existing scoring behavior and calibration limits; generated scorer inventories and report renderings remain under ignored `experiments/results/scoring-consolidation/`.
