# Research Synthesis Records

This directory contains curated, committed research-memory records for stage summaries, method-family status, method retirement/deprioritisation, Deep Research influence, and project direction changes.

These records are not generated experiment outputs. They are source-of-truth metadata used by `libreprimus research-synthesis` and anti-drift checks.

Raw reports, raw Discord logs, page images, generated result dumps, and local third-party artefacts must not be committed here.
