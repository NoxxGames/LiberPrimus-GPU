# Generated Alignment Outputs

Purpose: default output directory for Stage 0D transcript-line, Pastebin-alignment, page-boundary, glyph-variant, warning, and summary records.

What belongs here: generated JSON/JSONL outputs from local smoke runs.

What does not belong here: raw transcripts, raw Pastebin text, canonical corpus records, or committed experiment evidence.

Codex may write generated files here for validation, but generated outputs are ignored by Git and must not be staged in Stage 0D.
