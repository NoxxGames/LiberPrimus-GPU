# data/normalized/legacy-pastebin

## Purpose

Ignored generated output directory for legacy Pastebin parser records.

## What belongs here

Generated line-pair JSONL, anchor JSON, summary JSON, and warning JSONL files from local parser smoke runs.

## What does not belong here

Canonical corpus data, raw Pastebin text, committed generated outputs, databases, or solved-page claims.

## Codex modification policy

Codex may write generated outputs here during validation, but only this README and `.gitkeep` should be committed.

## Stage 0C restrictions

Generated records are non-canonical artefacts and must remain ignored unless explicitly promoted in a later stage.
