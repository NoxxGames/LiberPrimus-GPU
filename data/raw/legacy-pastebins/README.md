# data/raw/legacy-pastebins

## Purpose

Stores local legacy Pastebin text drops as immutable raw artefacts.

## What belongs here

User-supplied local `.txt` copies of legacy Pastebin sources, such as `58-Pages-In-Runes-With-Prime-Values-Pastebin.txt`, after lock metadata is created.

## What does not belong here

Canonical corpus files, generated normalized records, inferred page boundaries, reports, logs, or rewritten text copies.

## Codex modification policy

Codex may add README and `.gitkeep` placeholders and may move an untracked user-supplied local text file here without changing its bytes. Codex must not commit raw Pastebin text files.

## Stage 0C restrictions

Files here are non-canonical legacy LP2 rune/prime-value sources. Page boundaries are not finalized from this directory.
