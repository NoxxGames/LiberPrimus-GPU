# Raw Transcript Sources

Purpose: holds immutable local transcript source files used for provenance and later corpus-freeze review.

What belongs here: raw transcript files acquired from documented public sources, plus README and `.gitkeep` placeholders.

What does not belong here: normalized corpus files, generated alignment outputs, edited transcripts, OCR experiments, or solved-page claims.

Codex may add source-specific placeholder directories and metadata references, but must not modify raw transcript files in place. Stage 0D raw transcript files are ignored by Git and hash-locked under `data/locks/transcripts/`.
