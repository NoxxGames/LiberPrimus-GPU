# rtkd Master Transcription

Purpose: stores the local raw copy of the rtkd master Liber Primus transcription candidate.

What belongs here: `liber-primus__transcription--master.txt` when mirrored from the documented raw GitHub URL.

What does not belong here: normalized records, edited transcript variants, generated alignments, or page-image OCR output.

Codex may download the documented raw transcript if absent, but must not treat it as an active canonical corpus in Stage 0D. The raw file is ignored by Git; only lock metadata is committed.
