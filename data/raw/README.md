# data/raw

## Purpose

Raw evidence only.

## What belongs here

Future original images, transcripts, message extracts, and hash records that have explicit source metadata and SHA-256 locks.

## What does not belong here

Generated normalized data, derived text, edited images, search results, logs, local scratch files, or experiments.

## Codex modification policy

Codex must treat `data/raw/` as immutable. Do not modify or normalize files in place. Do not overwrite raw data.

## Stage 0A restrictions

Do not commit large or real raw corpus files in Stage 0A. Only this README and `.gitkeep` placeholders are expected.
